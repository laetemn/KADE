 Kade Arcade - Customizable USB Card - PC - XBOX - PS2 - PS3

 Easily build a PCB to assemble Arcade or Arcade Control.

 KADE is basically an Arcade adapter compatible with: PC, PSOne, PS2, PS3 and Xbox Classic. Other systems are supported using PS2 controller adapters and this
 includes Xbox 360.

 Through the Kade Loader application, you can customize which system you will program the control on, as well as map the buttons to your liking!
